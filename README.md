Find treasure
====

Linuxコマンド勉強用リポジトリ。

### Description

Linux初心者向けに基本的なコマンドを理解してもらうための練習リポジトリです。
以下の学習を目的としています。

- 相対パスの理解
- cd, ls, pwd, catコマンドの理解

### Usage

手元にCloneし、フォルダに移動してスタートしてください。

```
git clone https://github.com/sota1235/find_treasure.git
cd find_treasure
cat start_tour.txt
```
